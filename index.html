Haotian Wang

I am Haotian Wang, a third-year PhD researcher with experience applying machine learning and statistical modeling to large-scale, real-world data.

I build end-to-end ML systems for large-scale spatiotemporal data, integrating ground measurements and remote sensing to support robust, uncertainty-aware decision-making in hydrology, climate, and agriculture.

Publications & Projects

  • A Two-Stage Machine Learning Framework for Reliable Flood Inundation Mapping under Imbalanced Data

    Haotian Wang, Fangni Lei, Xinyi Shen, Qing Yang, Emmanouil N Anagnostou, Wade T Crow, Hyunglok Kim, Clara Chew

    Journal of Hydrology 2025

    Accurate and timely flood mapping is critical for disaster response and risk management, yet remains challenging due to sparse signals and highly imbalanced data. We develop a two-stage machine learning framework that explicitly tackles data imbalance by separating flood detection from extent estimation, enabling more reliable and scalable flood mapping from satellite observations.

  • Generative Vision Model for Flood Change Detection

    Faezeh Maghsoodifar, Md Shadman Sakib, Mohamed Mowafy, Haotian Wang, Sagy Cohen, Yinphan Tsang, Hamed Moftakhari, David F Muñoz, Patrick Ray, Fangni Lei

    CUAHSI 2025

    Instead of interpolating flood maps pixel-wise, this work adopts an image warping formulation that captures how flood extents evolve spatially across return periods. A deep learning model is used to learn constrained deformation fields under monotonic risk assumptions, providing a statistically grounded and computationally efficient way to reconstruct missing flood scenarios.

  • Quantifying Didelity-accuracy Trade-off in Operational Flood Mapping: A Comparative Analysis

    Md Shadman Sakib, Mohamed Mowafy, Haotian Wang, Faezeh Maghsoodifar, Sagy Cohen, Yinphan Tsang, David F Muñoz, Patrick Ray, Fangni Lei, Hamed Moftakhari

    CUAHSI 2025

    This project builds an automated data evaluation pipeline that standardizes heterogeneous datasets across formats, resolutions, and data types, enabling scalable and reproducible accuracy comparisons. The system integrates statistical evaluation, visualization, and reporting into a fully automated workflow, delivering both an interactive dashboard and a production-ready pipeline for continuous model and data assessment.

  • An A/B Evaluation Framework for Soil Moisture Products Using Statistical Error Decomposition

    Haotian Wang, Qiangqiang Yuan, Hongfei Zhao, Hongzhang Xu

    Journal of Hydrology 2022

    We design a three-arm A/B test to evaluate competing soil moisture estimation strategies across diverse environmental conditions. By leveraging a statistical error-decomposition model rather than direct pairwise comparison, the study isolates the strengths and weaknesses of each data source.
